Intersect

Description

Return a new DataFrame containing rows only in both this DataFrame and another DataFrame. This is equivalent to 'INTERSECT' in SQL.

Usage

## S4 method for signature 'DataFrame,DataFrame'
intersect(x, y)

intersect(x, y)

Arguments

x

A Spark DataFrame

y

A Spark DataFrame

Value

A DataFrame containing the result of the intersect.
